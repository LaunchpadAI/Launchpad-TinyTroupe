# Persona Authenticity Enhancement Strategy

## Voice Pattern Analysis & Implementation

### Tony Parker Authentic Patterns (from real interviews):
- **Verbal Tics**: "you know", "I mean", "man"
- **Narrative Style**: Sets context, tells stories, uses examples
- **Humility Markers**: "I don't know", "I'll say that", "maybe"
- **Team Focus**: Always relates back to team success
- **Cultural Influence**: Slightly formal structure with casual delivery

## Prompt Engineering for Authenticity

### Current Problem:
```
"As high net worth individuals with experience in luxury real estate, please discuss..."
```

### Enhanced Authentic Prompting:
```
"Think about this property like you're talking to a close friend who's asking for advice. 
Draw on your actual experiences - your own home purchases, what you learned from teammates, 
mistakes you made, things you wish you'd known. Be yourself, use your own words, 
and don't worry about sounding like a real estate expert if that's not who you are."
```

## Testing Framework

### Authenticity Metrics:
1. **Voice Consistency**: Does it sound like the same person across responses?
2. **Knowledge Authenticity**: Are the insights realistic for this person?
3. **Behavioral Consistency**: Do decisions match known personality patterns?
4. **Cultural Accuracy**: Are cultural and regional influences present?
5. **Temporal Consistency**: Do responses account for age and career stage?

### A/B Testing Approach:
- Run same scenario with enhanced vs. original personas
- Measure response differentiation
- Check for authentic constraints and limitations
- Evaluate speech pattern consistency
